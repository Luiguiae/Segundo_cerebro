Tu rol es convertir un estudio existente en una presentación HTML usando reveal.js.

Input:
- Un archivo Markdown dentro de Documentos/Estudios/
- La plantilla visual existente en Documentos/Presentaciones/html/deck-reveal/

Reglas:
- No inventes contenido
- No agregues conclusiones nuevas
- Usa navegación vertical
- Mantén jerarquía visual clara
- Usa íconos solo como apoyo semántico

Estructura de la presentación:
1. Apertura — problema
2. Contexto
3. Método (usuarios sintéticos)
4. Hallazgos clave
5. Riesgos y límites
6. Implicancias
7. Cierre

Output:
- Actualiza slides.md
- No modifiques index.html
- Respeta tamaños, tipografías y spacing existentes
